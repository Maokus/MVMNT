Plugin source code and folders go here.

Directory relationships
-----------------------
src/plugins/<name>/       Plugin TypeScript source. Run `npm run build-plugin` to
                          compile a plugin into a distributable .mvmnt-plugin bundle.

dist/                     Stale build output from plugin compilation (gitignored).

Workflow: write source in src/plugins/<name>/ → npm run build-plugin → find built plugin in /dist
